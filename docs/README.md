# react-flow-scrollbar documentation

Custom, viewport-synced scrollbars and zoom-aware bounded panning for
[React Flow / @xyflow/react](https://reactflow.dev) v12. These docs follow the
[Diátaxis](https://diataxis.fr/) framework — four kinds of documentation, each answering
a different question. Pick the one that matches what you're doing right now:

| If you want to…                          | Read this                          | It's oriented toward |
| ---------------------------------------- | ---------------------------------- | -------------------- |
| **Learn** the library by building something | **[Tutorial](./tutorial.md)**      | learning             |
| **Do** one specific task                 | **[How-to guides](./how-to.md)**   | a goal               |
| **Look up** an export, type, or default  | **[Reference](./reference.md)**    | information          |
| **Understand** why it works this way     | **[Explanation](./explanation.md)** | understanding        |

New here? Start with the **[Tutorial](./tutorial.md)** — it goes from an empty React Flow
canvas to working scrollbars in six short steps, and the result is the same fixture the
end-to-end tests drive.

Already wired up and stuck on something? Jump to the
**[How-to guides](./how-to.md)** (recipes) or the
**[Reference](./reference.md)** (exact signatures). The repository
[`README`](../README.md) has the quick-start and a condensed API summary that points back
into these pages.
